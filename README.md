# Merchy
#### Merchy is a web application for some of the most popular merchandise brands out there.
### Merchy is built in react

## Technologies Used
* React
* JavaScript
* Git
* GitHub
* Vercel

## Description 
_Merchy.com is a website for displaying some of the best merchandise out there. Merchy contains a fleet of Nike, Michael Jordan, Louis Vuitton and much more_

## Setup/Installation
1. Clone the repo
```sh
    git clone https://github.com/Emzzy241/merchy.git
```

2. Install all needed Project Dependencies and Packages
```sh
    npm install
```

3. Build & Run the Application
```sh
    npm run start
```
